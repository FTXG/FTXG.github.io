1、从应用商店下载小米运动App (现已更名为: Zepp Life)；

2、打开小米运动App，通过手机号注册登录；

3、登录之后，点击我的->第三方接入->绑定需要同步的第三方平台；

4、回到网页版，输入小米运动账号密码，提交步数即可同步至你绑定的所有平台。

注意：
1、小米运动非小米手机也可以安装；

2、账号密码建议使用不常用密码，以防信息泄露；

3、新注册的账号刚开始可能会提示密码错误，等一分钟左右即可；

4、网页刷取成功后，小米运动app会自动掉线。此时无需再打开小米运动，直接去看第三方平台的步数，已经同步好了。

5、如果网页上提示刷取成功，zepplife步数变化了但是第三方平台步数没有增加，请仔细查看上面蓝色字第三条。

如果绑定了第三方平台没有自动同步，以微信为例，解决方法如下：打开“Amazfit 跃我”公众号 → 左下角”产品中心“ → 排行榜，弹出运动数据后即可完成同步。

6、步数切勿贪多，建议4w左右即可，太多容易被平台检测。
